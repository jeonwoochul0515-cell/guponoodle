# 스마트콜 ARS 설정 (전화 문의 줄이기)

> ⚠️ 현재 번호 2~5가 OFF라 ARS가 작동 안 함 → **전부 ON**으로 켤 것.
> ⚠️ 현재 번호 4(주차)·번호 5(영업시간) 내용은 **사실과 다름** → 아래로 교체.

## 기본 멘트 (수정)
```
안녕하세요. 구포국수체험관입니다.
예약과 당일 방문 문의는 2번, 가격과 연령은 3번,
오시는 길과 주차는 4번, 영업시간은 5번,
상담원 연결은 1번을 눌러주세요.
```

## 번호 1 — 상담원 연결 (ON · 통화 연결)
```
상담원과 연결합니다.
```

## 번호 2 — 예약·당일 방문 (ON · 안내 + 문자 발송)
```
예약과 남은 자리는 네이버에서 '구포국수체험관'을 검색하시면
예약창에서 날짜·시간별로 실시간 확인하실 수 있습니다.
당일은 잔여 좌석이 있는 경우에 한해 현장 결제로 가능합니다.
예약 방법을 문자로 보내드립니다. 상담원 연결을 원하시면 1번을 눌러주세요.
```
(문자 내용 예: 네이버 예약 → '구포국수체험관' 검색 → 개인 예약창에서 예약. 문의 010-2608-5099)

## 번호 3 — 가격·연령 (ON · 안내)
```
체험은 어린이 한 명당 24,000원이며, 보호자는 무료로 함께 참여하실 수 있습니다.
36개월, 만 3세부터 권장합니다.
상담원 연결을 원하시면 1번을 눌러주세요.
```

## 번호 4 — 오시는 길·주차 (ON · 안내 + 문자 발송) ★내용 교체 필수
```
지하철 3호선 구포역 3번 출구에서 도보 3분 거리입니다.
전용 주차장은 없어 인근 유료 주차장을 이용해 주시고,
지정 주차장은 30분 주차가 지원됩니다.
자세한 길 안내를 문자로 보내드립니다. 상담원 연결을 원하시면 1번을 눌러주세요.
```
(문자 내용 예: 부산 북구 구포만세길 109 / 구포역 3번 출구 도보 3분 / 인근 유료주차장: 구포·대천·남선, 지정주차장 30분 지원)

## 번호 5 — 영업시간 (ON · 안내) ★내용 교체 필수
```
영업시간은 화요일부터 일요일까지 오전 10시부터 오후 6시까지이며,
매주 월요일은 휴무입니다.
개인 체험은 주로 주말과 공휴일에 운영되고, 평일은 단체 예약 위주입니다.
상담원 연결을 원하시면 1번을 눌러주세요.
```

## 통화연결음(업체소개) 멘트 — 연결 중 자동 안내
> 전화 연결되는 동안 재생. 늦은 분·당일 방문자에게 받기 전에 안내. 짧고 또렷하게.
```
안녕하세요, 구포국수체험관입니다. 전화해 주셔서 감사합니다. 예약 시간에 늦으실 것 같으면 회차 시작 후 15분까지 입장 가능하니 미리 말씀해 주세요. 입구는 1층 국수집 카운터 오른쪽 엘리베이터입니다. 당일 방문은 잔여 좌석이 있는 경우 현장 결제로 가능합니다. 곧 전화를 받을 예정이오니 잠시만 기다려 주세요.
```

### 영업 외 시간 통화연결음 (선택 · "전화 받을 수 없는 시간" 기능 ON)
```
안녕하세요, 구포국수체험관입니다. 지금은 영업시간이 아닙니다. 영업시간은 화요일부터 일요일 오전 10시부터 오후 6시까지이며, 월요일은 휴무입니다. 예약은 네이버에서 구포국수체험관을 검색해 주세요.
```

## 정리
- 모든 번호 ON → 자동응답으로 자리·가격·길·시간 문의가 걸러짐
- 번호 4·5 내용은 반드시 위 정확한 내용으로 교체(기존은 거짓 정보)
- 입력 후 'ARS 테스트'로 직접 들어보고 '저장'
