## B. 근거가 약한(검증 필요) 홈페이지 표현

홈페이지에만 있고 플레이스에서 확인 안 되는 값. 사실이면 유지, 아니면 수정.
- "건물 뒤편 무료 주차 약 10대" → **거의 확실히 오류(자체 주차장 없음)**. 삭제·교체 1순위.
- "15,000+ 가족이 선택한 체험" → 누적 방문자 수 근거 있는지 확인.
- "98% 아이가 좋아했어요" → 산출 근거 확인(없으면 리뷰 키워드 기반 표현으로 완화).
- 인근 관광지(구포시장·낙동강하구에코센터·화명수목원) → 사실이면 유지.

## C. 정책적으로 정할 것
- C1. 홈페이지(.lol) 예약 버튼을 **네이버 예약으로 일원화**할지(현재 플레이스로 예약 유도). "홈페이지 예약 안 받음, 플레이스로" 공지가 과거에 있었음 → 이 방침 유지가 맞는지 확정.
- C2. 부정 리뷰(예약자 확인·동선 안내) 대응. ① 1층 입구·엘리베이터 안내문 보강 ② 예약자 본인 확인 절차 정비 ③ 해당 리뷰에 정중한 답글. (운영 개선이라 콘텐츠와 별개지만 같이 메모)
