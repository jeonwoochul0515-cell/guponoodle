# 사이트 고도화 로드맵 (우선순위순)

> 기준. 비즈니스 목표 = **예약(전환)** + **검색 유입**. 임팩트 대비 노력으로 정렬.
> 표기. 🟢 내가 바로 가능 / 🟡 사장님 정보·계정 필요 / 🔴 결정 필요

## ⭐ 1순위 — 지금 당장 효과 + 신뢰/정직성 직결

### 1-1. 가짜 후기 → 진짜 네이버 후기로 교체 🟢🔴 (가장 시급)
- 현재 홈페이지 후기(지민맘·준서아빠 등)는 **실제가 아닌 예시로 지어낸 글**입니다. 방문자 신뢰를 떨어뜨리고, 구글·네이버가 **허위 후기로 감점**할 수 있습니다.
- 회원님은 **진짜 리뷰 2,026개·평점 4.84**가 있습니다. 실제 리뷰 문구(작성자 표기) 일부를 옮기거나, "네이버 리뷰 보러가기" 링크/별점 표시로 바꾸는 게 정직하고 강력합니다.

### 1-2. 전환 측정(GA4 이벤트) 🟢
- 지금 GA4는 방문수만 봅니다. **'예약하기' 클릭·전화 클릭·블로그/인스타 클릭을 이벤트로 추적**하면, 무엇이 예약으로 이어지는지 측정됩니다. "측정 안 되면 개선도 없다."

### 1-3. 구글 서치콘솔 등록 🟡
- 네이버 색인은 했지만 **구글 서치콘솔(Search Console)** 등록은 안 된 것으로 보입니다. 구글은 .lol 사이트를 빨리 잡아주니, 등록 시 색인·검색어·오류를 관리할 수 있습니다. (소유확인 메타태그/파일만 추가하면 됨)

### 1-4. '한눈에 보기' 예약 요약 카드 🟢
- 히어로 바로 아래에 **가격·시간·연령·운영·예약버튼**을 한 박스로. 스크롤 없이 핵심+예약 동선을 노출 → 전환↑. (앞서 제안했던 것)

## ⭐ 2순위 — 유입·노출 확장

### 2-1. 영어 페이지(외국인 관광객) 🟢🟡
- 부산은 외국인 가족여행 수요가 크고, 네이버 플레이스도 외국인 예약을 받습니다. **영문 버전(/en)** 을 만들면 "Busan kids activity / noodle making" 검색을 잡을 수 있습니다. (번역·구조는 제가, 운영 결정은 사장님)

### 2-2. AggregateRating 구조화 데이터 🟢🔴
- 실제 평점 4.84·리뷰 2,026을 **schema.org Review/AggregateRating**으로 넣으면 구글 검색에 ★별점이 노출될 수 있습니다. 단, **반드시 실제 수치 + 정책 준수**(자체 페이지 자가평점은 신중히). 1-1과 함께 진행해야 안전.

### 2-3. 실제 사진 갤러리 + 짧은 영상(릴스) 🟡
- 체험 단계별 실제 사진 갤러리, 30초 제면 영상 임베드 → 체류시간·전환 크게 상승. (사진/영상은 사장님, 배치는 제가)

### 2-4. 네이버 애널리틱스 추가 🟢
- 타깃이 네이버이므로, 네이버 애널리틱스(wcs) 스크립트를 넣으면 네이버 유입·검색어를 별도로 볼 수 있습니다.

## ⭐ 3순위 — 성능·품질·신뢰 마감

### 3-1. 광고(AdSense) 재검토 🔴
- 예약 유도가 목적인 사이트에 **구글 애드센스 광고가 2곳** 있습니다. 광고 수익보다 **예약 전환·신뢰를 깎는 손해**가 클 수 있습니다. 제거 또는 위치 조정 권장(사장님 판단).

### 3-2. 성능 추가 최적화 🟢
- 폰트(Pretendard·Noto Serif) **self-host 또는 preload**, Font Awesome **전체 JS → 사용 아이콘 SVG만**, 전용 **OG 공유 이미지**(텍스트 포함) 제작. → 로딩·공유 카드 품질↑.

### 3-3. 보안 헤더 🟢
- `vercel.json`에 CSP·X-Content-Type-Options 등 보안 헤더 추가(무해, 신뢰도/평가 소폭↑).

### 3-4. 지도 임베드 정확도 점검 🟢
- 구글 지도 iframe이 정확히 체험관을 가리키는지 확인(좌표는 맞지만 embed 핀 확인 필요). 네이버 지도 링크 병행도 고려.

### 3-5. 문의 채널(네이버 톡톡/카카오) 🟡
- 즉시 문의 버튼(네이버 톡톡 등)을 달면 예약 전 질문을 잡아 이탈을 줄입니다.
