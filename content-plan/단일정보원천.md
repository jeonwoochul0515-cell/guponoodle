## 4. 체험 상품·가격
### (1) 개인 체험 — `<개인> 제면놀이+민속맷돌체험+편백놀이`
- 가격. 24,000원
- 대상. 10인 이하 개인 고객 전용
- 권장 연령. 36개월(만 3세) 이상
- 소요시간. **1시간 30분** (✅확정 2026.05.29). 홈페이지 60분은 오기 → 수정 대상
- 예약. 반드시 개인 예약창으로 (네이버 예약)
- 결제. 네이버페이. 카드/현금/계좌이체 원할 시 전화 문의
- 운영 안내. 회차당 적정 인원 미충족 시 해당 회차 취소될 수 있음
- 누적 방문자 리뷰. 1,896

### (2) 단체 체험 — `단체예약전용 (제면놀이+민속맷돌체험+편백놀이+어린이국수)`
- 가격. 24,000원
- 대상. **15인 이상**(협의 가능) 어린이집·학교·기관 단체 전용 (✅확정 2026.05.29). 홈페이지 "10인 이상"은 수정 대상
- 예약. 전용 네이버 단체 예약창 (날짜·시간 선택 후 직원 확인 전화)
- 결제. 현장 당일 결제 (카드·계좌이체·현금)
- 주의. 개인이 단체창에 예약하면 자동 취소

## 5. 체험 진행 흐름 (리뷰·블로그에서 확인된 실제 동선)
1. 3층 대기실 인사 → 구포국수 역사 간단 설명
2. 2층 이동 → 민속 맷돌체험(통밀 갈기) + 편백나무 놀이존
3. 3층 재이동 → 국수 만들기
   - 밀가루 반죽 (아이가 직접 물 붓고 반죽) → 숙성 대기
   - 대기 중 수제비 만들기 (자색고구마 반죽, 밀대로 밀어 꽃모양 찍기)
   - 숙성 반죽으로 제면기 돌려 국수 뽑기
- 제공. 일회용 앞치마·위생모자
- 결과물. 직접 만든 **소면·수제비·코인육수(육수알)** 밀키트로 포장 귀가
- 특징. 미취학 아동은 부모 도움 필요 / 초등학생은 대부분 스스로 가능 / 실내 진행(우천 무관)

## 6. 어른(보호자) 관련

## 7. 주차 (✅확정 2026.05.29)
- 자체 주차장 **없음**. 자가용 이용 시 인근 유료주차장 이용
- **주차 지원**. 지정(제휴) 주차장에 한하여 **30분 지원**
- 지정/인근 주차장 3곳
  - 구포주차장 — 부산 북구 구포만세길 104
  - 대천주차장 — 부산 북구 구포만세길 112
  - 남선주차장(국민렌트카) — 부산 북구 낙동대로1714번길 9
- 요금(주차장 자체 기준, 약). 기본 3,000원 / 2시간 이용 시 약 4,500원 — 정확한 증가 단위는 주차장별 상이하므로 표기 시 "약"으로
- ⚠️ 홈페이지 "건물 뒤편 무료 주차 약 10대"는 사실과 정반대 → **1순위 수정**
