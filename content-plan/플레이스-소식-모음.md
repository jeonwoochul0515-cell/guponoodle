# 플레이스 소식 글 모음 (주 1회 로테이션)

> 스마트플레이스 → 마케팅(또는 업체정보) → 소식 → 새 소식 작성. 사진 1장 이상 함께 올리면 노출이 더 좋습니다.

## 소식 ① 주말 체험 예약 안내 (유형: 알림)
```
[이번 주말 제면체험 예약 안내]
구포국수체험관 주말 개인 체험 회차를 예약 받습니다 🍜
🕙 회차 10:00 / 10:30 / 13:00 / 13:30
👶 36개월(만 3세)부터 · 보호자 무료 · 약 1시간 30분
🎁 직접 만든 소면·수제비·코인육수 밀키트 포장
예약은 네이버 예약에서 '구포국수체험관' 검색!
```

## 소식 ② 평일 단체 모집 (유형: 알림)
```
[평일 단체 체험 모집]
어린이집·유치원·학교 단체(15인 이상)의 평일 제면체험을 받습니다.
국수 만들기 + 민속 맷돌 + 편백놀이로 구성된 1시간 30분 프로그램이며, 인원은 협의 가능합니다.
문의·예약 ☎ 010-2608-5099 또는 네이버 단체 예약창.
```

## 소식 ③ 밀키트 포인트 (유형: 소식)
```
[체험이 집까지 이어져요]
구포국수체험관은 아이가 직접 만든 국수를
소면·수제비·코인육수 밀키트로 포장해 드립니다.
집에 가서 끓여 먹으면 "내가 만든 국수"라며 한 그릇 뚝딱!
체험의 추억이 저녁 식탁까지 이어집니다 🍜
```

## 소식 ④ 휴무 안내 템플릿 (유형: 알림 — 날짜만 바꿔 사용)
```
[휴무 안내]
구포국수체험관은 매주 월요일 정기 휴무입니다.
○월 ○일(○)은 (사유) 휴무합니다.
이용에 참고 부탁드리며, 주말·공휴일 체험은 정상 운영됩니다 🙏
```

## 소식 ⑤ 후기 감사 + 방문 유도 (유형: 소식)
```
[2,000개가 넘는 후기, 감사합니다 💛]
구포국수체험관을 찾아주신 가족 여러분 덕분에
방문자 리뷰 2,000건, 평점 4.84를 기록했습니다.
"아이가 너무 좋아했어요" "집에서 또 끓여 먹었어요" 한마디 한마디가 큰 힘이 됩니다.
지하철 3호선 구포역 3번 출구 도보 3분, 주말 체험으로 만나요!
```

## 등록 팁
- 주 1회 꾸준히. ①②는 매주 반복, ③④⑤는 사이사이 섞기.
- 소식마다 사진 1장 이상 첨부(없으면 대표 체험 사진 재활용).
- 휴무·일정 변경이 생기면 ④를 그때그때 올려 정확성 유지.
