# 도보 코스 분석 — 구포국수체험관 + 낙동강 강변

> 코스. (체험관/구포역) ↔ 감동나룻길 리버워크 ↔ 금빛노을브릿지 ↔ 화명생태공원 ↔ 구포시장
> 한 줄 평. **네 지점이 모두 구포역 인근에 모여 있어, 차 없이 반나절에 도는 가족 도보 코스로 최적.** 특히 체험(실내) → 강변 산책 → 일몰 → 전통시장으로 흐름이 자연스럽다.

## 1. 위치 관계 (핵심)
- **감동나룻길 리버워크** — 구포역과 화명생태공원을 잇는 보행전용교(320m, 2024.12 개통). **구포역에서 도보 약 5분.**
- **금빛노을브릿지** — 낙동강 위 383m 보행교(2022 개통). **감동나룻길에서 도보 약 5분.** 서향이라 일몰·야경 명소. 승강기 3대로 유모차·휠체어 접근 가능.
- **화명생태공원** — 낙동강변 대형 공원(관리면적 3.03㎢, 길이 7.74km). 두 다리가 공원으로 이어짐. 입장·주차 무료.
- **구포시장** — 구포역 인근(구포만세거리로 연결), 400년 전통시장.
> ※ 공원이 7.74km로 길어 **전 구간을 다 걷는 게 아니라**, 구포역 쪽(남측) 입구 일대만 도는 것이 핵심. (덕천역·금곡동 쪽은 공원 북측 다른 입구이니 혼동 주의.)

## 2. 추천 동선 & 구간별 소요(추정)
| 구간 | 거리/시간(대략) | 메모 |
|------|----------------|------|
| 체험관(1층 국수 식사) → 구포역 → 감동나룻길 입구 | 도보 5~10분 | 출발 |
| 감동나룻길 리버워크 건너기 | 320m, 7분 | 강 위 보행교, 포토존 |
| → 금빛노을브릿지 | 도보 5분 | 이동 |
| 금빛노을브릿지(일몰 감상) | 383m, 15~30분 | 코스 하이라이트 |
| 화명생태공원 산책 | 자유(30분~1시간) | 메타세쿼이아·억새·연꽃, 자전거 |
| → 구포시장·구포만세거리 | 도보 15~20분 | 마무리 |
| 구포시장 구경·간식 | 30분~ | 전통시장 |

**총 소요 대략 3~4시간(반나절), 순수 도보 약 3~4km.** 공원 산책 길이에 따라 가감.

## 3. 동선 방향 — 어느 순서가 좋나
- **일몰 보려면(추천).** 체험 → 식사 → 감동나룻길 → **금빛노을브릿지에서 일몰** → (어두워지면) 마무리. → 일몰 1~1.5시간 전 출발이 핵심.
- **시장 마무리형.** 낮에 공원·다리 산책 → 구포시장에서 간식·장보기로 마무리(말씀하신 순서).
- 체험관·구포시장이 모두 구포역 쪽이라 **출발·도착이 같은 권역 = 사실상 순환 코스**.

## 4. 아이 동반 적합성
- **장점.** 전 구간 평지·강변, 보행교 승강기로 **유모차 OK**. 공원 넓어 뛰놀기 좋고 자전거 대여 가능.
- **주의.** ① 강변·공원은 **그늘이 적어 여름 한낮 더위·자외선** 주의(모자·물 필수) ② 총 도보 3~4km라 **아주 어린 유아는 유모차 권장** ③ 일몰 후 어두워지면 강변 안전 유의 ④ 공원 화장실 위치 미리 확인.

## 5. 베스트 시기
- **봄.** 유채·벚꽃·튤립(6.5만 구) / **가을.** 억새·코스모스 / **여름.** 연꽃 습지(단, 더위 주의) / **연중.** 일몰·야경.

## 6. 체험관 연계 추천(반나절 모델)
**오후 1시 또는 1시 30분 체험 예약 → 1층 구포국수 식사 → 감동나룻길 → 금빛노을브릿지(일몰) → 화명생태공원 → 구포시장.**
오전 회차면 공원·시장을 먼저 돌고 오후 늦게 일몰로 마무리해도 좋음.
