# 플레이스 사진 촬영 가이드 (현재 4/120장)

> 사진은 플레이스 노출·예약 전환에 큰 영향을 줍니다. 아래 목록 순서대로 채우면 됩니다.
> 블로그 기둥글의 [사진] 자리에도 같은 컷을 재활용하세요.

## 1순위 — 대표컷 (먼저 채울 것)
- [ ] **제면기에서 하얀 국수가 쭈욱 나오는 순간** (이 체험의 상징, 대표사진으로)
- [ ] 아이가 국수/반죽을 들고 환하게 웃는 클로즈업
- [ ] 완성된 밀키트(소면·수제비·코인육수) 포장 모습

## 2순위 — 찾아오는 길 (외지 방문객용)
- [ ] 건물 외관·간판 (정면)
- [ ] 구포역 3번 출구에서 체험관까지 가는 길 (이정표처럼 2~3컷)
- [ ] 입구·엘리베이터 위치 (※ 부정리뷰에서 지적된 동선, 명확히 보여주면 컴플레인 예방)

## 3순위 — 체험 단계별
- [ ] 민속 맷돌로 통밀 가는 모습
- [ ] 편백나무 놀이방
- [ ] 밀가루 반죽하는 손
- [ ] 꽃모양 수제비 만들기
- [ ] 제면기 돌리는 과정
- [ ] 일회용 앞치마·위생모 착용 컷

## 4순위 — 공간·신뢰 요소
- [ ] 깔끔한 내부 전경 (2·3층 체험실)
- [ ] 역사관/전시 도구 (1950~60년대 제면 도구)
- [ ] 가격·안내판
- [ ] 단체 체험 현장 (단체 예약 유치용)

## 5순위 — 1층 국수 식당
- [ ] 1층 식당 내부
- [ ] 물국수·잔치국수·김밥 등 메뉴 사진

## 촬영 팁
- 낮에 밝은 자연광에서, 흔들림 없이. 어두운 사진은 피하세요.
- **아이 표정**이 들어간 컷이 반응이 좋습니다(타인 아이 얼굴은 보호자 동의 필요).
- 가로·세로 모두 찍어두면 플레이스·블로그·인스타에 골라 쓰기 좋습니다.
- 다른 방문객 얼굴이 크게 나온 사진은 개인정보 문제로 피하세요.
